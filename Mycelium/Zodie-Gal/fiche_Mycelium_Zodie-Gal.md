## Composante technique
De ce que j'ai compris sur le projet. Le jeu va être sur une page web, alors pour le projet, un serveur web sera nécessaire. Un serveur web c'est « ...un serveur web est un ordinateur qui stocke les fichiers qui composent un site web (par exemple les documents HTML, les images, les feuilles de style CSS, les fichiers JavaScript) et qui les envoie à l'appareil de l'utilisateur qui visite le site. »

Source pour cette information: [L'information](https://developer.mozilla.org/fr/docs/Learn/Common_questions/Web_mechanics/What_is_a_web_server)
